# College Management System

A comprehensive campus & examination management system built with React and TypeScript.

## Features

- **Admission & Student Registration**: Online application forms, document upload, applicant tracking
- **Attendance Management**: Multiple entry modes, real-time dashboards, automated notifications
- **Examination & Results**: Exam scheduling, result computation, grade cards, certificates
- **Finance & Fee Management**: Fee structures, online payments, receipts, financial reports
- **Role-Based Access Control**: Admin, Staff, Student, Parent roles with appropriate permissions
- **Reporting & Analytics**: Real-time dashboards, exportable reports, performance charts
- **Support Modules**: Library, Store/Inventory, Transport/Hostel, HR management

## Getting Started

### Prerequisites

- Node.js 18+ and npm/yarn

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

## Tech Stack

- React 18
- TypeScript
- Material-UI (MUI)
- React Router
- Recharts (for analytics)
- Vite

## Project Structure

```
src/
├── components/     # Reusable UI components
├── pages/         # Page components
├── contexts/      # React contexts for state management
├── types/         # TypeScript type definitions
├── utils/         # Utility functions
├── services/      # API services
└── routes/        # Route configuration
```
